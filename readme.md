# Sistema de Registro de Visitas al Aula MakerSpace

Este programa permite registrar visitas de estudiantes al Aula MakerSpace.

Para cada estudiante se almacena:

* Nombre
* Curso
* Motivo de la visita

# Funcionalidades

* Registrar al menos 3 estudiantes.
* Mostrar la lista completa de visitas.
* Contar cuantas visitas hubo por curso.
* Mostrar el primer estudiante registrado.

# Temas aplicados

* Input y Print.
* Funciones.
* Estructuras de control (if y for).
* Listas.
* Diccionarios.

# Lenguaje utilizado
Python
